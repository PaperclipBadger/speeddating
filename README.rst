speeddating
===========

Simulator
---------

Simulate the effectiveness of various speed dating algorithms on various cohorts:

.. code:: bash

    poetry run python -m speeddatingsim

Website
-------

Website for facilitating speed dating sessions, using the recommender algorithm:

.. code:: bash

    poetry run python -m speeddatingsim.server

Creates a sqlite database in the :code:`speeddatingsim` directory.
